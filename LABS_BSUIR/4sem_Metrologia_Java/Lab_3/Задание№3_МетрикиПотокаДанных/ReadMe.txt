Тема задания: "Метрики сложности потока данных"

1- Изучить метрики сложности потока данных (файл МетрикиПотокаДанных_Теория.docx).

2- Взять код программы в консольном режиме не менее чем на 200-300 строк (например, найти в Интернет) на указанном преподавателем языке программирования. 

3- Написать свою программу (на любом языке программирования), которая будет анализировать программу из пункта 2. На основе анализа программа должна рассчитать и вывести на экран метрики потока данных:
 - спены идентификаторов и суммарный спен программы (результаты должны быть сведены в таблицу, аналогичную таблице 8);
 - полная метрика Чепина и метрика Чепина ввода/вывода (результаты должны быть сведены в таблицу, аналогичную таблице 9 с соответствующими пояснениями по распределению переменных по группам).

Для защиты:
- программа, считающая метрики Холстеда + анализируемая программа;
- отчет согласно примерам в файле с теорией, но для своей анализруемой программы, составленный в электронном виде и распечатанный.

В процессе защиты необходимо продемонстрировать корректно работающую программу. Программа должна адекватно реагировать на вносимые при тестировании изменения. Знать определения ВСЕХ метрик сложности потока данных и формулы ИХ расчета.
