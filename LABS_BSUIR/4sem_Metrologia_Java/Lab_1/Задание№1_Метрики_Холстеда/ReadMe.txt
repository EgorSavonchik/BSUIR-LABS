Тема задания: "Метрики размера программ"


1- Изучить метрики размера программ на примере метрики Холстеда (файл Метрика_Холстеда_Теория.docx).

2- Взять код программы в консольном режиме не менее чем на 200-300 строк (например, найти в Интернет) на указанном преподавателем языке программирования. 

3- Написать свою программу (на любом языке программирования не в консоли), которая будет анализировать программу из пункта 2. На основе анализа программа должна рассчитать метрики Холстеда согласно примеру в файле из пункта 1. Результат работы программы: вывести на экран 6 основных метрик Холстеда и 3 расширенные в удобном для просмотра виде (например, в таблицу). 

4 - составить отчет (один на двоих) в электронном виде и распечатать. Содержание отчета: титульный лист, фрагмент кода на одну страницу формата А4 (в коде должен присутствовать основной синтаксис заданного языка по максимуму), скрин с результатами работы программы, таблица с самостоятельно вручную рассчитанными метриками. 

Для защиты:
- программа, считающая метрики Холстеда + анализируемая программа;
- отчет согласно примерам в файле с теорией, но для своей анализруемой программы (выбранной части программы), составленный в электронном виде и распечатанный.

В процессе защиты необходимо продемонстрировать корректно работающую программу. Программа должна адекватно реагировать на вносимые при тестировании изменения. Знать определения ВСЕХ метрик Холстеда и формулы ИХ расчета.
